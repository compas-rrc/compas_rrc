Zone

class compas_rrc.Zone[source]

Bases: object

Describes the valid zone data definitions.

There are two types of zones: fine and fly-by. If zone is FINE, the movement terminates as a stop point, and the program execution will not continue until robot reach the stop point. For all other zones, the movement terminates as a fly-by point, and the program execution continues about 100 ms before the robot reaches the zone.
